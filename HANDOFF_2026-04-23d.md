## THE ONE THING TO BUILD
Route PNG saves through main thread. Thread encodes PNG to bytes in memory (no file I/O),
puts bytes into _save_queue. Main thread writes bytes to disk. Zero Defender exposure.

## ROOT CAUSE (CONFIRMED)
Windows Defender blocks BOTH file opens AND file writes from background threads.
- File opens: solved by numpy pre-decode on main thread
- File writes: NOT YET SOLVED — thread hangs on cv2.imwrite() and _probe_file.write()
- Result: probes stop at PROBE 6, zero PNGs written, Resolve hangs on close

## WHAT WORKS
- BRAW → TIFF export: works (6 TIFFs written on main thread)
- PIL decode to numpy on main thread: works ("Decoded 6 frames to numpy" in log)
- chroma_hint_gen warmup on main thread: works
- Model load: works ("Model loaded!")
- Thread starts: works (alive=True)
- Probe 3-6 fire: works (direct file write, 6 probes before Defender clamps)
- Probes 7+ never fire: thread HANGS on _probe_file.write() (Defender blocking write)

### Change 2: Replace save_output call in thread (near line 1901-1904)
CURRENT CODE (lines 1901-1904 approx):
```python
if fg is not None and mt is not None:
    op = od / f"CK_{cn}_{pr:06d}.png"
    save_output(fg, mt, op, settings["export_format"])
    ofs.append(str(op))

```

REPLACE WITH:
```python
if fg is not None and mt is not None:
    op = od / f"CK_{cn}_{pr:06d}.png"
    # Encode PNG to bytes IN MEMORY — no file I/O, no Defender block.
    _fmt = settings["export_format"]
    _m = mt[:, :, 0] if len(mt.shape) == 3 else mt
    if _fmt == 0:
        _fb = cv2.cvtColor((fg * 255).astype(np.uint8), cv2.COLOR_RGB2BGR)
        _au = (_m * 255).astype(np.uint8)
        _img = cv2.merge([_fb[:,:,0], _fb[:,:,1], _fb[:,:,2], _au])
    elif _fmt == 1:
        _img = (_m * 255).astype(np.uint8)
    else:
        _img = cv2.cvtColor((fg * 255).astype(np.uint8), cv2.COLOR_RGB2BGR)
    _ret, _buf = cv2.imencode('.png', _img)
    if _ret:
        _save_queue.put(("save", str(op), _buf.tobytes()))
        ofs.append(str(op))
    pr += 1
```

    try:
        item = _save_queue.get_nowait()
        if item[0] == "save":
            _, path_str, png_bytes = item
            try:
                with open(path_str, 'wb') as _sf:
                    _sf.write(png_bytes)
            except Exception as _se:
                log(f"Save error {path_str}: {_se}")
