## ROOT CAUSE (FULLY SOLVED)

Two bugs, both fixed:

1. **PNGs not writing** — The BRAW render queue `time.sleep(1.0)` wait loop blocked the Fusion event loop long enough to permanently kill the PollTimer. The `_save_queue` architecture (thread encodes, main thread writes via timer) never fired.
   - **Fix:** Added synchronous BRAW path. When `_braw_frames_decoded` is populated (all frames in memory), skip the thread entirely. Process frames directly on main thread using `save_output()`. Same as single-frame mode. Timer not needed.

2. **Off-by-one frame count** — Resolve's timeline end is exclusive. `src_end = ss + (out_f - cs)` was missing the last frame.
   - **Fix:** `src_end = ss + (out_f - cs) + 1`
