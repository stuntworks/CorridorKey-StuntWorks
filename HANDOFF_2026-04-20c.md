### 1. SAM2 architecture is wrong
Currently SAM2 REPLACES the chroma key hint. It should be a GARBAGE MATTE on top of chroma key.

Correct approach:
- Chroma key runs → soft alpha hint with good edges
- SAM2 mask applied as garbage matte: where SAM2=background → force alpha to 0
- Where SAM2=foreground → leave chroma key result alone

Fix in generate_alpha_hint():
```python
elif settings["alpha_method"] == 1:

    chroma_hint = AlphaHintGenerator(screen_type=settings["screen_type"]).generate_hint(frame)
    # Get SAM2 mask (from alpha.png or propagation)

    # Garbage matte: force background areas to black
    import numpy as np
    result = chroma_hint.copy()
```
